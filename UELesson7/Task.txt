Створити Pause-Menu для гри. Меню повинно:
1. Відкриватися та закриватися поверх гри (із напівпрозорим фоном) на натиск кнопки Tab (зупинка гри вже реаалізована); 
2. Вмикати курсор для взаємодії із меню;
3. Мати 2 вікна для перемикання:
	a. Загальне меню із кнопкою продовження гри і кнопка завершення гри;
	b. Вікно налаштувань із зміною імені гравця та прапорцем (які уже є властивостями персонажа), чи показувати ім’я у грі.
Для перемикання між вікнами використати кнопки перемикання по краях.

4. Для відображення імені гравця реалізувати 3D віджет, який завжди обернений до користувача.
